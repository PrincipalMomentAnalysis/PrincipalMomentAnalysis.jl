PrincipalMomentAnalysis.pma — Function
pma(A, G; nsv=6)

Computes the Principal Moment Analysis of the matrix A (variables × samples) using the sample adjacency graph G. Set nsv to control the number of singular values and vectors returned. Returns a PMA struct.

See also PMA.

source
PrincipalMomentAnalysis.PMA — Type
PMA <: Factorization

The output of pma representing the Principal Moment Analysis factorization of a matrix A. If F::PMA is the factorization object, U, S, V and Vt can be obtained via F.U, F.S, F.V and F.Vt such as A ≈ U * Diagonal(S) * Vt'.

See also pma.
